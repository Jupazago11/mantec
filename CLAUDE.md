# CLAUDE

Antes de modificar codigo en este repositorio, leer primero:

1. [AGENTS.md](/home/jupazago/Documentos/mantecv1/mantec/AGENTS.md)
2. [ANALISIS_SISTEMA_LARAVEL.md](/home/jupazago/Documentos/mantecv1/mantec/ANALISIS_SISTEMA_LARAVEL.md)
3. [ARRANCAR_LOCAL.txt](/home/jupazago/Documentos/mantecv1/mantec/ARRANCAR_LOCAL.txt) cuando aplique entorno local o servicios.

La regla operativa es simple: no tocar codigo sin hacer preflight de contexto, alcance, riesgos, seguridad, pruebas y documentacion.


Secuencia obligatoria:

Leer contexto -> inspeccionar -> definir alcance -> implementar -> autorizar y asegurar -> probar -> verificar interfaz -> revisar efectos colaterales -> documentar -> revisar diff -> informar resultado.

El informe final siempre debe cubrir:

- Implementado
- Archivos afectados
- Validaciones
- Resultado
- Pendientes

No ocultar pruebas fallidas ni presentar exito parcial como exito completo.
